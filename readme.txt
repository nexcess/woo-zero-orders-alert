Get email alerts when your store has zero sales for the previous day.

== Description ==

Getting no sales for an entire day is usually a sign of a different problem. Site down? Payment gateway not working? Zero Orders Alert for WooCommerce will send an email to the store owner when zero sales are recorded for the previous day.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `woo-zero-orders-alert` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. That's it.

== Frequently Asked Questions ==

= What is this for? =

To make sure you don't miss a problem with your store.

= How do I set it up? =

Activate it, that's it. The plugin runs in the background using WP-Cron, so there are no settings to configure.

= Can I change how a feature works / looks? =

There are filters inside the plugin allowing developers to modify certain aspects of the alerts.
